#  Keen_Keeper

##  Project Name

**Keen_Keeper**

##  Description

Keen_Keeper is a smart activity tracking and interaction management app that helps users keep track of their communications such as calls, texts, and video interactions in an organized timeline. It provides filtering options and visual insights to better understand user activity.

##  Technologies Used

* React.js
* JavaScript (ES6+)
* Tailwind CSS
* Context API (State Management)
* Recharts (for data visualization)

##  Key Features

*  **Filterable Timeline**
  Users can filter activities like calls, texts, and video interactions with a single click.

*  **Activity Analytics**
  Visual representation of interaction data using charts for better insights.

*  **Real-time State Management**
  Efficient data handling using React Context API without external storage.
